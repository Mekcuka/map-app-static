## Обзор

Приложение позволяет:

- Размещать объекты нефтегазовой инфраструктуры на карте (OpenStreetMap)
- Соединять объекты трубопроводами с расчётом протяжённости
- Редактировать геометрию трубопроводов (точки изгиба)
- Перемещать объекты с автоматическим обновлением связей
- Управлять видимостью слоёв
- Экспортировать/просматривать параметры объектов и труб

### Скриншот интерфейса

```
┌──────────────┬─────────────────────┬──────────────┐
│  Левая       │                     │  Правая      │
│  панель      │       Карта         │  панель      │
│              │                     │              │
│  • Режимы    │   [Объекты]         │  • Список    │
│  • Добавить  │      ↕              │    объектов  │
│  • Редактир. │   [Трубы] ────●───  │  • Список    │
│  • Слои      │          ↑ точка    │    труб      │
│              │         изгиба      │              │
└──────────────┴─────────────────────┴──────────────┘
```

## Установка и запуск

### 1. Клонирование и установка

```bash
cd map-app-react
npm install
```

### 2. Запуск dev-сервера

```bash
npm run dev
```

Приложение доступно по адресу: **http://localhost:5173**

### 3. Сборка для продакшена

```bash
npm run build
```

Результат сборки в папке `dist/`.

### 4. Предпросмотр продакшен-сборки

```bash
npm run preview
```

### Управление состоянием

Состояние централизовано в `App.jsx`:

| State | Тип | Описание |
|-------|-----|----------|
| `objects` | `Array<Object>` | Все объекты на карте |
| `connections` | `Array<Connection>` | Все трубопроводы |
| `objIdSeq` | `number` | Счётчик ID объектов |
| `pipeIdSeq` | `number` | Счётчик ID труб |
| `mode` | `'view' \| 'edit'` | Текущий режим |
| `addType` | `string` | Тип добавляемого объекта |
| `addClick` | `boolean` | Флаг режима размещения |
| `pipeFrom` | `Object \| null` | Начало трубы |
| `pipeTo` | `Object \| null` | Конец трубы |
| `pipeWaypoints` | `Array<[lat,lng]>` | Промежуточные точки трубы |
| `showLayers` | `{map, obj, pipe}` | Видимость слоёв |

### Оптимизация производительности

- `useMemo` для маркеров и труб — пересоздание только при изменении данных
- `React.memo` для `PipeLine` и `WaypointMarker`
- `requestAnimationFrame` throttle для `mousemove`
- `useRef` для callbacks — стабильные ссылки без пересоздания

## Руководство пользователя

### Режимы работы

#### 👁️ Просмотр (view)
- Навигация по карте (перетаскивание, зум)
- Просмотр информации об объектах (клик → popup)
- Выбор объектов и труб

#### ✏️ Редактор (edit)
- Перетаскивание объектов мышью
- Перетаскивание точек изгиба труб
- Добавление точек изгиба кликом на трубу
- Удаление точек изгиба (ПКМ → подтверждение)

#### 📍 Размещение (place)
- Активируется при выборе типа объекта
- Курсор показывает превью объекта
- Клик на карту размещает объект

### Объекты

#### Типы объектов

| Тип | Иконка | Название | Параметр | Единица |
|-----|--------|----------|----------|---------|
| `wellpad` | 🛢️ | Куст скважин | Уровень добычи | т.н/год |
| `upsv` | ⚙️ | УПСВ | Производительность | т.н/год |
| `kns` | 💧 | КНС | Производительность | т.ж/год |
| `cps` | 🏭 | ЦПС | Производительность | т.н/год |
| `node` | 🔵 | Узел | — | — |

#### Добавление объекта

1. В левой панели выберите тип объекта
2. Курсор на карте покажет превью
3. Кликните на карту для размещения

#### Редактирование объекта

1. В правой панели наведите на объект
2. Нажмите **✏️ Редактировать**
3. Измените название и/или параметр
4. Нажмите **Сохранить**

#### Перемещение объекта

1. Переключитесь в режим **✏️ Редактор**
2. Зажмите левую кнопку мыши на объекте
3. Перетащите в нужное место
4. Отпустите — трубы автоматически подтянутся

#### Удаление объекта

- В правой панели нажмите **🗑️ Удалить** или
- Клик на объект → в popup нет кнопки удаления (только через панель)

### Трубопроводы

#### Создание трубопровода

**Способ 1: Между двумя объектами**
1. Выберите **🔧 Трубопровод**
2. Кликните на первый объект (начало)
3. Кликните на второй объект (конец)
4. Введите параметры в модальном окне
5. Нажмите **Создать трубопровод**

**Способ 2: С промежуточными точками**
1. Выберите **🔧 Трубопровод**
2. Кликните на объект-начало
3. Кликайте на карту для добавления изгибов
4. Кликните на объект-конец

**Способ 3: Завершение узлом (двойной клик)**
1. Начните рисовать трубу (клик на объект)
2. Добавьте изгибы кликами
3. **Двойной клик** на карту → создаётся узел, труба завершается

**Способ 4: Через существующую трубу**
1. Выберите **🔧 Трубопровод**
2. Кликните на линию существующей трубы
3. Создаётся узел на трубе — начало новой трубы
4. Кликните на конечный объект

#### Параметры трубопровода

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Название | Имя трубопровода | "Труба N" |
| Внешний ⌀ | Внешний диаметр, мм | 500 |
| Внутренний ⌀ | Внутренний диаметр, мм | 480 |

#### Редактирование трубопровода

1. В правой панели нажмите **✏️ Редактировать** у трубы
2. Измените название или диаметры
3. Нажмите **Сохранить**

#### Удаление трубопровода

- В правой панели нажмите **🗑️ Удалить**

---

### Редактирование геометрии

#### Добавление точки изгиба

1. Переключитесь в режим **✏️ Редактор**
2. Кликните на линию трубопровода
3. Точка изгиба добавляется в ближайшую к курсору позицию

#### Перемещение точки изгиба

1. В режиме **✏️ Редактор** найдите синюю точку на трубе
2. Зажмите и перетащите точку
3. Труба автоматически перерисуется

#### Удаление точки изгиба

1. В режиме **✏️ Редактор** нажмите **ПКМ** на точке изгиба
2. Подтвердите удаление

#### Сброс геометрии

1. Откройте редактирование трубы (✏️ в правой панели)
2. Нажмите **↺ Сбросить геометрию**
3. Труба станет прямой линией между объектами

## API компонентов

### App.jsx

Главный компонент, управляет всем состоянием приложения.

**State:**
```javascript
{
  objects: Array<{id, type, name, center: [lat, lng], params}>,
  connections: Array<{id, name, from, to, pts: [[lat,lng],...], od, idm}>,
  mode: 'view' | 'edit',
  // ...
}
```

**Ключевые функции:**

| Функция | Описание |
|---------|----------|
| `addObject(lat, lng, type)` | Добавить объект |
| `handlePipeClick(obj)` | Обработка клика при создании трубы |
| `confirmPipe(data)` | Создать трубу из модального окна |
| `moveObject(id, lat, lng)` | Переместить объект + обновить трубы |
| `moveWaypoint(pipeId, idx, lat, lng)` | Переместить точку изгиба |
| `removeWaypoint(pipeId, idx)` | Удалить точку изгиба |

### MapView.jsx

Компонент карты с Leaflet.

**Props:**
```javascript
{
  objects, connections, selObjId, setSelObjId,
  mode, addType, addClick, pipeFrom, pipeWaypoints,
  showLayers, onMapClick, onMapDblClick, onObjectClick,
  onPipeLineClick, onMoveObject, moveWaypoint, removeWaypoint
}
```

**Внутренние компоненты:**

| Компонент | Описание |
|-----------|----------|
| `DraggableMarker` | Маркер объекта с drag-n-drop |
| `PipeLine` | Линия трубопровода (React.memo) |
| `WaypointMarker` | Точка изгиба с drag-n-drop |
| `MapEvents` | Обработчики событий карты |

### SidebarLeft.jsx / SidebarRight.jsx

Панели управления.

**Props SidebarRight:**
```javascript
{
  objects, connections, selObjId, selPipeId,
  setSelObjId, setSelPipeId, mode,
  deleteObject, deletePipe, openEditPipe, openEditObject
}
```

### Modal.jsx / ObjectModal.jsx

Модальные окна.

**Props Modal:**
```javascript
{
  isOpen, onClose, onSubmit,
  pipeFrom, pipeTo, pipeIdSeq,  // для создания
  editData, setEditData, onResetGeom, isEdit  // для редактирования
}
```

## Известные ограничения

1. **Нет сохранения** — данные теряются при обновлении страницы
2. **Нет экспорта/импорта** — нельзя сохранить проект в файл
3. **Toast-уведомления** — заменены на `alert()`
4. **Нет валидации** — не проверяется корректность ввода параметров
5. **Нет истории** — нет undo/redo
