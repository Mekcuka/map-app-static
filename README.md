# 🗺️ Карта объектов и трубопроводов

Интерактивное веб-приложение для размещения объектов на карте с привязкой к координатной сетке, созданием и редактированием трубопроводов между ними.

## 🚀 Быстрый старт

Просто откройте файл `index.html` в браузере — сервер не требуется!

## ✨ Функциональность

| Функция | Описание |
|---------|----------|
| 📍 Объекты | Квадраты 100×100 метров с привязкой к сетке |
| 🔗 Трубопроводы | Связи между объектами с характеристиками |
| ✏️ Редактирование | Перемещение объектов, изменение трубопроводов |
| 📐 Координаты | WGS 84 (широта/долгота) |
| 👁️ Слои | Вкл/выкл карты, объектов, трубопроводов, сетки |

## 🛠️ Характеристики трубопроводов

Каждый трубопровод содержит:
- **📏 Протяжённость** — рассчитывается автоматически по координатам (формула гаверсинусов)
- **⌀ Внешний диаметр** — задаётся при создании (мм)
- **⌀ Внутренний диаметр** — задаётся при создании (мм)

## 📖 Использование

### Добавление объекта
1. Введите имя (опционально)
2. Нажмите **"🖱️ Добавить кликом"** → кликните на карту

### Создание трубопровода
1. Нажмите **"🔗 Добавить трубопровод"**
2. Кликните на первый объект
3. Кликните на второй объект
4. Введите диаметры в появившихся окнах

### Редактирование
1. Нажмите **"✏️ Редактирование"**
2. Выберите объект в списке или кликните на карте
3. Перетащите объект мышью или измените координаты в панели
4. Кликните на трубопровод → **"Редактировать"** для изменения диаметров

## 🛠️ Технологии

- **Leaflet 1.9.4** — интерактивные карты
- **OpenStreetMap** — тайлы карты
- **Vanilla JavaScript** — без фреймворков
- **Чистый CSS** — адаптивный дизайн
