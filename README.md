# 🗺️ Карта трубопроводов

Интерактивное веб-приложение для планирования и редактирования трубопроводных сетей.

## 🚀 Быстрый старт

Просто откройте файл `index.html` в браузере — сервер не требуется!

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📍 **Объекты** | Квадраты 100×100 м с автонумерацией |
| 🔗 **Трубопроводы** | Связи с параметрами (диаметры, протяжённость) |
| ✏️ **Редактор** | Перемещение объектов и точек изгиба |
| 📐 **Координаты** | WGS 84 (широта/долгота) |
| 👁️ **Слои** | Управление видимостью карты, объектов, труб |

## 🛠️ Характеристики трубопроводов

Каждый трубопровод содержит:
- **Название** — автонумерация: «Труба 1», «Труба 2»…
- **📏 Протяжённость** — авто-расчёт по координатам (м/км)
- **⌀ Внешний диаметр** — задаётся при создании (мм)
- **⌀ Внутренний диаметр** — задаётся при создании (мм)
- **📐 Точки изгиба** — редактируемая геометрия

## 📖 Использование

### Режимы работы

| Режим | Описание |
|-------|----------|
| **👁️ Просмотр** | Навигация по карте, просмотр информации |
| **✏️ Редактор** | Перемещение объектов, редактирование геометрии труб |
| **🔗 Труба** | Создание нового трубопровода между объектами |

### Добавление объекта
1. Откройте режим **👁️ Просмотр**
2. Введите название (опционально) и координаты
3. Нажмите **«📍 Добавить по координатам»** или **«🖱️ Добавить кликом»**

### Создание трубопровода
1. Переключитесь в режим **🔗 Труба**
2. Кликните на первый объект
3. Кликните на второй объект
4. В модальном окне укажите параметры (диаметры)
5. Нажмите **«Создать трубопровод»**

### Редактирование геометрии трубы
1. Режим **✏️ Редактор**
2. Кликните на линию трубы → добавьте точку изгиба
3. Перетащите голубой маркер для изменения формы
4. Двойной клик по маркеру → удалить точку
5. Клик на трубу → **«✏️ Редактировать»** → параметры

## 🛠️ Технологии

- **Leaflet 1.9.4** — интерактивные карты
- **OpenStreetMap** — тайлы
- **Vanilla JavaScript** — без зависимостей
